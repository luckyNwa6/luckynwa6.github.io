前言

选择器优先级：

默认(继承) < 通配符选择器 < 标签选择器 < 类选择器 < ID选择器 < 行内样式 < !important

选择器示例

深度选择器

深度选择器用于修改第三方组件（如 Element-UI）的内部样式。当 <style> 使用 scoped 时，不加深度选择器则无法修改子组件的样式。

使用方式：

  • .el-form >>> .el-form-item__error（Vue2）
  • ::v-deep（Vue2）
  • :deep()（Vue3 推荐）
